# 🧮 Windows Calculator Automation using Pywinauto, Pytest & Allure

This project demonstrates **Windows desktop application automation** using **Pywinauto** on the built-in **Windows Calculator (UWP app)**.  
It includes **standalone automation**, **pytest-based test cases**, and **Allure report generation**.

---

## 📁 Project Structure

calculator_testing/
│
├── calculator_app.py # Standalone automation script
├── test_calculator.py # Pytest-based automation tests
├── allure-results/ # Generated during pytest execution
├── allure-report/ # Generated Allure HTML report
└── README.md
